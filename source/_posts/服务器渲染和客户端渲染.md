# 服务器渲染和客户端渲染
<!-- more -->
```js
1、浏览器端渲染路线：
请求一个 html。
服务端返回一个 html。
浏览器下载 html 里面的 js/css 文件。
等待 js 文件下载完成。
等待 js 加载并初始化完成。
由 js 代码向后端请求数据（ajax/fetch）。
等待后端数据返回。
客户端从无到完整地，把数据渲染为响应页面。 
2、服务端渲染路线：
请求一个 html。
服务端请求数据
服务器初始渲染。
服务端返回已经有正确内容的页面。
客户端请求 js/css 文件。
等待 js 文件下载完成。
等待 js 加载并初始化完成。
客户端把剩下一部分渲染完成。
```
